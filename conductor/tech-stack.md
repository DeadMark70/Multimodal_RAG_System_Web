# Technology Stack

## Frontend
- **Framework:** React (TypeScript) + Vite
- **State Management:** Zustand, React Query (TanStack Query v5)
- **UI Component Library:** Chakra UI, Framer Motion, Emotion
- **Visualization:** React Force Graph 2D, Recharts, @xyflow/react
- **HTTP Client:** Axios

## Backend
- **Framework:** FastAPI + Uvicorn
- **Language:** Python

## Data & Storage
- **Database:** Supabase (PostgreSQL)
- **Vector Database:** FAISS
- **Authentication:** Supabase Auth

## AI & Processing
- **LLM:** Google Gemini API (gemini-3.0-flash, gemma-3-27b-it)
- **OCR:** Local Marker / Datalab API
- **Core Logic (RAG, Graph):** Python FastAPI
